# Opti Maps

## Sobre

Opti Maps é um aplicativo desenvolvido em Flutter para organizar, otimizar e navegar por rotas de entrega.

O objetivo é oferecer uma ferramenta completa para:

- Motoboys
- Motoristas de entrega
- Representantes comerciais
- Técnicos de campo
- Empresas de logística

---

## Tecnologias

- Flutter
- Dart
- SQLite
- Repository Pattern

Planejado:

- Riverpod
- Google Maps
- Google Places
- OCR (Leitura de notas)
- QR Code
- Inteligência Artificial
- Backup em Nuvem
